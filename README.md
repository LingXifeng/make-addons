# Make Addons (Web)

Minecraft Bedrock Edition Addon 制作工具 - Web 版本

复刻自 MAM (Minecraft Addon Maker) Android 应用，纯前端实现，无需后端。

## ✨ 功能

- ⚔️ **武器模块** - 剑/弓/弩/盾/箭/锤 (25 个属性)
- 🛡️ **防具模块** - 头盔/胸甲/护腿/靴子 (4 件套)
- 🍎 **食物模块** - 自定义食物
- 📦 **导出 .mcaddon** - 打包 Behavior Pack + Resource Pack
- 🎨 **自定义纹理** - 上传 PNG 纹理或选择预设图标
- 💾 **本地保存** - 项目数据自动保存到浏览器
- 📋 **实时预览** - JSON 实时预览，支持复制

## 🛠️ 技术栈

- React + TypeScript
- Vite
- Zustand (状态管理)
- JSZip (打包导出)

## 🚀 开发

```bash
npm install
npm run dev
```

## 📦 构建

```bash
npm run build
```

## 🎮 支持的 Minecraft 版本

- Format version: 1.21.100+
- Min engine version: 1.21.100

## 📄 许可

本项目仅用于学习和个人使用。
